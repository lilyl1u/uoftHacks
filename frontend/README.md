# Frontend

React + TypeScript frontend built with Vite.

## Quick Start

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Open `http://localhost:3000`

## Environment Variables

Optional: Create `.env` file with `VITE_API_URL` (defaults to `http://localhost:3001/api`)

## Features

- Authentication (Login/Signup)
- User Profile with personality types
- Washroom Map (placeholder for map integration)
- Rating System
- Add Washroom functionality
