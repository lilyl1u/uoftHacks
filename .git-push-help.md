# Git Push with Personal Access Token

Since Git is hanging when trying to prompt for credentials, use one of these methods:

## Method 1: Embed Token in URL (Temporary)

```bash
git remote set-url origin https://lilyl1u:YOUR_TOKEN@github.com/lilyl1u/uoftHacks.git
git push
```

Replace `YOUR_TOKEN` with your actual Personal Access Token.

**Note:** This stores the token in plain text in `.git/config`. Remove it after pushing:
```bash
git remote set-url origin https://github.com/lilyl1u/uoftHacks.git
```

## Method 2: Use SSH (Recommended)

1. Set up SSH key if you haven't:
   ```bash
   ssh-keygen -t ed25519 -C "your_email@example.com"
   cat ~/.ssh/id_ed25519.pub
   ```
   Copy the output and add it to GitHub: https://github.com/settings/keys

2. Change remote to SSH:
   ```bash
   git remote set-url origin git@github.com:lilyl1u/uoftHacks.git
   git push
   ```

## Method 3: Use Git Credential Manager

```bash
brew install --cask git-credential-manager
git config --global credential.credentialStore keychain
git push
```

Then enter your PAT when prompted.
